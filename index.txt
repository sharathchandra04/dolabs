Task 1: Introduction to Linux Commands
Description: In this task, you will learn about essential Linux commands related to Navigation such as cd, ls, and mkdir.
Prerequisites: None
Subtasks:
Navigate to the root directory of your system using the cd command.
List the contents of the root directory using the ls command.
Create a new directory named ubuntu_files using the mkdir command.
Enter the newly created directory using the cd command.
List the contents of the current directory using the ls command.
Task 2: Exploring Ubuntu File Structure
Description: In this task, you will explore the Ubuntu file structure and learn about different directories.
Prerequisites: None
Subtasks:
Navigate to the /etc directory using the cd command.
List the contents of the /etc directory using the ls command.
Explore the contents of the /etc directory and identify the purpose of some common configuration files.
Navigate back to the root directory using the cd command.
Enter the /var directory using the cd command.
List the contents of the /var directory using the ls command.
Task 3: Creating and Managing Directories
Description: In this task, you will practice creating and managing directories in Ubuntu.
Prerequisites: None
Subtasks:
Create a new directory named documents in the root directory using the mkdir command.
Enter the documents directory using the cd command.
Create three subdirectories named work, personal, and projects using the mkdir command.
List the contents of the documents directory to verify the creation of subdirectories.
Navigate to the work directory using the cd command.
Create a new text file named work_notes.txt using a text editor like nano or vim.
